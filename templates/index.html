About


ISEED

In our present information society, online (social) media are important sites of debate and discussion on a range of current affairs such as climate change. As such, these (social) media platforms house a trove of relevant information for the study of cultural and societal conflicts. Harvesting and analyzing this digital information, however, can be challenging.

ISEED-AREA

ISEED-AREA (ARgument Extractor and Analyzer) is a research tool that can extract and analyze arguments from online debates. It contains a set of functions for extracting causal statements and other forms of argumentation from (noisy) textual data, like short posts from social media. It includes two types of argument extractions:

  • Relation extractor: extracts cause-effect relations from a sentence using a set of regular expressions; no NLP tool is employed, so the function is unaware of Part-Of-Speech or Dependency-Relations among words. This function was designed to be used with short texts from social media.
  • If-then extractor: extracts if-then relations from a sentence using a set of regular expressions; no NLP tool is employed, so the function is unaware of Part-Of-Speech or Dependency-Relations among words. This function was designed to be used with short texts from social media. It supports six languages, namely: English, Italian, Spanish, French, German, and Polish, with English default.
ISEED-AREA also includes the possibility to scrap data directly from social platforms according to a given set of parameters, process, and download it.


The Pipeline

An ISEED-AREA pipeline consists of three steps. The first step extracts an argument from a given dataset. The second phase process, analyze, the extracted arguments and prepare for visualization. The third phase visualize the processed data using different plots.
