The input data, CSV file, to the pipeline or API must contain two-column data. The first column is an author-id, which can be anonymous or null. The second column is a text, content to be processed. For an example dataset, please look here

To use the pipeline, follow the following steps:

Click on the pipeline from the menu bar.

Select the dataset and upload it:

Click Add Dataset.

Select the type of argument extractor, IF-THEN Extractor, and data source upload.

Select the language of the file.

Then upload file.

Extract an argument:

Click Extract argument.

Click on the export CSV if you want to export raw extracted content.




To generate a word cloud of extracted words, click the word cloud and plot it to visualize them.




Click on download to download the word cloud image.




To generate a word frequency of extracted words, click the word frequency and plot it to visualize them.




Click on the download plot as png to download the word frequency plot.
